# Mini Oscyloskop TFT

Prosty oscyloskop zbudowany na Arduino i wyświetlaczu TFT przy użyciu biblioteki TFT_eSPI.

## Funkcje

- wyświetlanie przebiegu sygnału w czasie rzeczywistym
- siatka pomiarowa
- marker napięcia (Y)
- marker czasu (X)
- tryb delta do pomiaru różnicy napięcia lub czasu
- zatrzymanie przebiegu (freeze)

## Użyte elementy

- Arduino
- wyświetlacz TFT
- potencjometry
- przyciski
- przewody połączeniowe
